We can also delete properties from objects like this:

```js

delete ourDog.bark;

```

Example:

```js

const ourDog = {

  "name": "Camper",

  "legs": 4,

  "tails": 1,

  "friends": ["everything!"],

  "bark": "bow-wow"

};

  

delete ourDog.bark;

```
