You can easily increment or add one to a variable with the ++ operator.

```js

i++;

```

is the equivalent of

```js

i = i + 1;

```

Note: The entire line becomes i++;, eliminating the need for the equal sign.

  

Same for decreasing by 1. E.g.

```js

i--; == i = i - 1;

```
